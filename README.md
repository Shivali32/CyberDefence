# Cyber Defence
FA25-BL-CSCI-B649-8705

## Overview
- Contains coursework for CSCI-B649: Cyber Defence (Fall 2025).
- Covers practical and theoretical cybersecurity concepts.
- Focuses on hands-on labs and analytical assignments.
- Emphasizes real-world security mechanisms and defenses.

---

## Course Topics
- Cryptography and encryption
- Hashing and password security
- Authentication and access control
- Intrusion detection systems
- Web security and XSS
- Steganography
- Reverse engineering and binary analysis
- Network and system defense

## Assignments
### Assignment 1: Cryptography and Security Fundamentals
- Analysis of cryptographic constructions and security guarantees.
- Evaluation of confidentiality, integrity, authentication, and non-repudiation.

### Assignment 2: Network Security and IDS Analysis
- Firewall rule analysis and IDS performance evaluation.
- Probabilistic reasoning applied to intrusion detection.
